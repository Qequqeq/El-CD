# Лабораторная работа №1

## Описание
Эта лабораторная работа посвящена **первичному знакомству с разработкой на программируемых логических интегральных схемах (ПЛИС)** с использованием платы **Tang Nano 9K** и интегрированной среды разработки **GOWIN EDA**. Основная задача -- реализация простого бегущего огня (running light) на светодиодах платы, что позволяет освоить базовый цикл разработки FPGA-проектов: от написания кода до прошивки устройства. 

## Цель
Освоить полный цикл работы в **GOWIN EDA**, включая:
- Создание нового проекта для FPGA GW1NR-9 (Tang Nano 9K).
- Написание и отладку кода на языке **Verilog**.
- Проведение симуляции с использованием тестбенча.
- Синтез, размещение и трассировку (P&R).
- Генерацию bitstream-файла и прошивку на реальное оборудование.

## Файлы проекта
Репозиторий содержит ключевые исходные файлы лабораторной работы:
- **`runner.v`** — основной модуль, реализующий логику бегущего огня с использованием счетчика для создания видимой задержки.
- **`runnertb.v`** — тестбенч для функциональной симуляции: генерирует тактовые сигналы, импульсы сброса и сохраняет результаты в VCD-формат для анализа в .vcd формате для дальнейшего анализа.

## Инструкция по воспроизведению
1. Установите **GOWIN EDA** (версия 2026 или новее) с официального сайта.
2. Создайте новый проект, выбрав устройство **GW1NR-C** (Tang Nano 9K).
3. Импортируйте файлы `runner.v` и `runnertb.v` в проект.
4. Выполните симуляцию тестбенча для проверки логики.
5. Назначьте пины в соответствии со схемой платы, выполните синтез и сгенерируйте bitstream.
6. Подключите Tang Nano 9K по USB и прошейте устройство через встроенный программатор.
**Быстрый запуск без сборки:**
openFPGALoader -m -b tangnano9k lab1.fs
