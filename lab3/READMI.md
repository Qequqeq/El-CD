# Лабораторная работа №3

## Описание
В третьей лабораторной работе реализован **генератор псевдослучайных чисел** на основе **линейного регистра сдвига с обратной связью (LFSR)**. Сгенерированное 8‑битное значение отображается в шестнадцатеричном формате на двух семисегментных индикаторах с временным мультиплексированием. Плата **Tang Nano 9K** генерирует новое случайное число с частотой ~1 Гц, а индикаторы динамически переключаются с частотой ~1 кГц, создавая эффект одновременного свечения.

## Цель
Изучить принципы построения генераторов псевдослучайных чисел на LFSR, освоить динамическую индикацию на семисегментных дисплеях с мультиплексированием.

## Файлы проекта
- **`top.v`** -- главный модуль, управляющий частотой обновления LFSR, мультиплексированием индикаторов и отображением данных.
- **`lfsr8.v`** -- 8‑битный LFSR с параметризуемым начальным значением (SEED) и полиномом обратной связи.
- **`hex_to_7seg.v`** -- дешифратор шестнадцатеричного полубайта в 7‑сегментный код (активный высокий уровень, общий катод).
- **`genrator_tb.v`** -- тестбенч для проверки LFSR.
- **`lab3.fs`** -- готовый битовый файл для прошивки ПЛИС.
- **`lab3.cst`** -- constraints-файл с назначением выводов.

## Инструкция по воспроизведению
1. Установите **GOWIN EDA** (версия 1.9.11 или новее) с официального сайта.
2. Создайте новый проект, выберите устройство: **GW1NR9-С** (Tang Nano 9K).
3. Добавьте в проект файлы `top.v`, `lfsr8.v`, `hex_to_7seg.v`.
4. При необходимости запустите симуляцию тестбенча `genrator_tb.v` (в GOWIN или iverilog) для проверки работы LFSR.
5. Откройте редактор ограничений и импортируйте `lab3.cst` (либо назначьте пины вручную согласно схеме).
6. Выполните синтез (Synthesize), размещение и трассировку (Place & Route).
7. Сгенерируйте битовый файл (Generate Bitstream) -- получите `lab3.fs`.
8. Подключите плату Tang Nano 9K через USB, запустите программатор (Programmer) из GOWIN EDA или используйте **openFPGALoader**.

## Быстрый запуск без сборки (с готовым `lab3.fs`)
```bash
openFPGALoader -m -b tangnano9k lab3.fs
```
