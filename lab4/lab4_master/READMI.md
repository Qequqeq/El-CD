# SPI-мастер (ведущее устройство)

## Описание
Модуль `top_master` генерирует псевдослучайные 8‑битные числа с помощью LFSR и периодически отправляет их по протоколу SPI (режим 0, MSB first). Три светодиода отображают состояние линий `CS`, `SCLK` и `MOSI` для отладки.

## Цель
Разработать SPI-мастер с параметризованным битрейтом, генерацией данных и корректным формированием сигналов CS и SCLK.

## Файлы проекта
- **`top_master.v`** -- верхний модуль: таймер обновления, LFSR, SPI-мастер.
- **`spi_master_simple.v`** -- параметризуемый SPI-мастер с конечным автоматом.
- **`lfsr8.v`** -- генератор псевдослучайных чисел.
- **`top_tb.v`** -- тестбенч для симуляции.
- **`lab4_master.cst`** -- constraints-файл.

## Инструкция по воспроизведению
1. Установите **GOWIN EDA**.
2. Создайте проект для устройства **GW1NR9-C**.
3. Добавьте файлы `top_master.v`, `spi_master_simple.v`, `lfsr8.v`.
4. Импортируйте `lab4_master.cst` (или добавье вручную).
5. Выполните синтез, размещение, трассировку и сгенерируйте битовый файл.
6. Прошейте плату.

## Быстрый запуск
```bash
openFPGALoader -m -b tangnano9k lab4_master.fs
```
