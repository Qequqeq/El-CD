# Лабораторная работа №2

## Описание
Вторая лабораторная работа посвящена использованию встроенных кнопок и светодиодов отладочной платы **Tang Nano 9K**. Реализован **двоичный счётчик** на 6 светодиодах (отображает значения от 0 до 63). Управление осуществляется двумя кнопками:
- **Кнопка 1** (KEY1) -- увеличивает значение счётчика на 1 по фронту.
- **Кнопка 2** (KEY2) -- уменьшает значение счетчика на 1 по фронту.
- **Обе кнопки одновременно** -- сбрасывают счётчик в 0.

## Цель
Освоить работу с пользовательскими входами (кнопки) и выходами (светодиоды) на ПЛИС GW1NR-9 (Tang Nano 9K). Научиться обрабатывать нажатия кнопок с устранением дребезга (аппаратная синхронизация и детекция фронта) и реализовать простой счётчик с отображением состояния в двоичном коде.

## Файлы проекта
- **`top.v`** -- основной модуль, реализующий двоичный счётчик с управлением от двух кнопок (инкремент по KEY1, декремент по KEY2, сброс при одновременном нажатии KEY1+KEY2).
- **`lab2.cst** -- constraints-файл с назначением выводов для платы Tang Nano 9K:
- **`lab2.fs** -- готовый битовый файл (bitstream) для непосредственной прошивки ПЛИС.

## Инструкция по воспроизведению
1. Установите **GOWIN EDA** (версия 1.9.11 или новее) с официального сайта.
2. Создайте новый проект, выберите устройство: **GW1NR-C** (Tang Nano 9K).
3. Добавьте в проект файлы `top.v`.
4. При необходимости запустите симуляцию тестбенча для проверки логики.
5. Откройте **FloorPlanner** или редактор ограничений и импортируйте `lab2.cst` (либо назначьте пины вручную).
6. Выполните синтез (Synthesize), размещение и трассировку (Place & Route).
7. Сгенерируйте битовый файл (Generate Bitstream) -- получите `lab2.fs`.
8. Подключите плату Tang Nano 9K через USB, запустите программатор (Programmer) из GOWIN EDA или используйте **openFPGALoader**.

## Быстрый запуск без сборки (если используется готовый `lab2.fs`)
```bash
openFPGALoader -m -b tangnano9k lab2.fs
```
